# COM 355 Vibe Coding Assignment - Schedulr.AI
An application that can create optimal study plans for you based on your school schedule. 

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1EgLBjG-0Cp8U6-EJU0n-1XSQiHCOfk_S?fullscreenApplet=true

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
